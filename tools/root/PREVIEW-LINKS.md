# The Joint — Local preview

## One-click (recommended)

**Double-click `preview.bat`** or run **`npm run preview:all`** from repo root.  
Starts the hub (3999), website (4000), and app (4001). Keep all windows open.

## Individual previews

| Project   | URL | Command |
|-----------|-----|--------|
| **Website** | http://localhost:4000 | `npm run start:website` |
| **App**     | http://localhost:4001 | `npm run start:app` |

**Preview hub** (page with both links): `npm run preview:hub` → http://localhost:3999  
Or double-click **open-preview-hub.bat** to open the hub from file; then start the two servers above in separate terminals.
